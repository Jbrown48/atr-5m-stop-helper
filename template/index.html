How to use this on a 5-minute chart

Example bar (last completed 5m): close = {{ "%.2f"|format(result.entry_price) }}, low = {{ "%.2f"|format(result.swing_low) }}, high = {{ "%.2f"|format(result.swing_high) }}.

For a long setup:
- Use the last 5-minute swing low as your structure level.
- Place the stop {{ "%.1f"|format(result.k_atr) }} × 5m ATR below that swing low.
→ Suggested stop ≈ {{ "%.2f"|format(result.stop_long) }}, which is {{ "%.2f"|format(result.dist_long) }} points under the example close.

For a short setup:
- Use the last 5-minute swing high as your structure level.
- Place the stop {{ "%.1f"|format(result.k_atr) }} × 5m ATR above that swing high.
→ Suggested stop ≈ {{ "%.2f"|format(result.stop_short) }}, which is {{ "%.2f"|format(result.dist_short) }} points above the example close.

Position sizing idea:
- Decide your dollar risk per trade (e.g. $1,000).
- Divide that by the stop distance in points to get max share size.
Example: if the long stop is {{ "%.2f"|format(result.dist_long) }} points away, max size ≈ 1000 / {{ "%.2f"|format(result.dist_long) }} shares (or equivalent notional in options).
